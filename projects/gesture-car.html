Gesture Car

An IoT-powered RC car controlled in real time by hand gestures, using an MPU6050 accelerometer and peer-to-peer ESP-NOW communication between two ESP32 modules.

Project Overview

Gesture Car enables real-time wireless control of an ESP32-powered vehicle using hand movements. One ESP32 module reads tilt data from an MPU6050 accelerometer worn by the user, then transmits directional commands via the ESP-NOW protocol to a second ESP32 mounted on the car, which drives four motors accordingly.

The project demonstrates embedded systems programming, sensor interfacing, peer-to-peer wireless communication, and motor control — all built without relying on cloud infrastructure.
