Memory King

A Flutter-based mobile card-matching game with real-time Firebase backend, user authentication, profile management, and competitive leaderboard scoring.

Project Overview

Memory King challenges players to match cards as quickly and accurately as possible across progressively harder difficulty levels. The app reflects a full-stack mobile architecture: a Flutter UI layer communicates with Firebase Authentication for login/signup, Firestore for real-time score storage, and Firebase Analytics for user behaviour tracking.

Built as the final project for the Mobile Application Development course at Princess Sumaya University for Technology.
