Resume Builder

A multipage web application for creating professional resumes with live preview, user authentication, and persistent database storage.

Project Overview

Resume Builder is a full-stack web application that lets users create polished, professional resumes through a guided multi-step form with a live side-by-side preview. The backend is powered by PHP and MySQL, handling authentication, session management, and secure data persistence. The frontend uses HTML5, CSS3, JavaScript, and jQuery for interactive, responsive behavior.

Built as the final project for Webpage Design and Internet Programming (12243) at Princess Sumaya University for Technology, Spring 2024/2025.
